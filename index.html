How to use

Nbdev creates python modules from notebooks, so if you clone the repo you can call exported functions using the AoC_2022 package.

  1. Get the code. If using the GitHub CLI:
gh repo clone ctwardy/AoC_2022
  1. Install the package locally so AoC_2022 is in your namespace.
pip install -e '.[dev]'

Then you can do something like this:

import AoC_2022.day2 as day2

with open("data/day2_input.txt") as f:
    data = [day2.rps_decode(x.strip()) for x in f.readlines()]
day2.score_strategy(data)
